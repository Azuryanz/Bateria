Sobre

Este projeto foi feito para praticar HTML, CSS e JavaScript, com base no projeto original da B7Web.


Para fins de prática, a estrutura HTML e CSS foi feita por conta própria, apenas olhando o layout original. De mesma forma, novos elementos JavaScript foram adicinados, como a barra de volume e o este painel lateral de informações.


Para mais informações, acesse a vídeo-aula do projeto original: B7Web - Bateria

Tutorial

Barra de Volume


Um usuário pode controlar o volume dos áudios com base nesta barra.


Enquanto o slider não estiver encostado no lado esquerdo da barra, as teclas serão brevemente coloridas de um tom de ciano quando seu som for reproduzido.


Quando o slider estiver encostado no lado esquerdo as teclas serão brevemente coloridas de um tom de vermelho ao tentar reproduzir seu som.


Teclado


Para reproduzir um som, você pode clicar em um dos nove botões da tela (Q W E A S D Z X C) ou apertar as respectivas teclas em seu teclado.


O comportamento da bateria depende da posição do interruptor Ouvir/Mutar, como explicado acima.


Área da Composição


No campo de texto logo abaixo dos nove botões, um usuário pode digitar uma sequência de teclas para ser tocada. Há uma espera de 250ms entre a reprodução do som associado de cada tecla.


Quaisquer teclas digitadas além das nove anteriores serve apenas para ganhar tempo entre uma tecla válida e outra.


Após finalizar uma composição, basta clicar no botão Tocar para reproduzí-la.
